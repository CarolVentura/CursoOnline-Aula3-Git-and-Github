Planning da aula Git e Github 

23/03 - das 09h as 17h 

Objetivo - > Entender o que é e pra que usamos git, github e linha de comando. 
             Instalação e conta no github. 
             Entender comandos básicos do git, criar respositório no github, 
             subir repositório local, clonar repositório remoto, 
             Entender git pull e pull request. 

-------------------------

1 -> apresentação 
         chamada e só pedir pras meninas dizerem de onde são. Explicar objetivo da aula. 

2 -> Conceitos/história git, github e linha de comando 
         checar versão, fazer instalação(caso necessária), brincar um pouco na linha de comando, conta no github 

3 -> Subir projeto local(apenas com o readme.md) no github. 
         entender comandos: git init, git add . , git status, git commit -m'',git remote add origin, git push. 
         entender estágios dos arquivos (vendo o git status)

almoço

        clonar repositório remoto(meu)

        criar branch com nome delas, fazer alteração, subir. 

6 -> git pull 
        explicar conceito e mostrar em uso 

7 -> pull request 
        explicar conceito e deixar elas fazerem pr.
